2. Permission to use on Top Huatulco.
3. Credit line: owner-supplied, photographer name if desired, or “Top Huatulco.”
4. Confirmation that visible people/license plates are acceptable or can be cropped.
5. Preferred page target if the same photo could fit multiple routes.
